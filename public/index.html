Regras e Limites Operacionais da API

Busca em Lotes Sequenciais

O programa consulta a API oficial de contribuintes do Ambiente de Dados Nacional em /DFe/{NSU}. A busca é feita por NSU, em blocos sequenciais, salvando localmente cada XML retornado no lote e avançando até não haver mais documentos compatíveis com os filtros.

Prevenção de Consumo Indevido (Erro 656)

Consultas repetidas e muito rápidas podem gerar bloqueio temporário por consumo indevido. Para reduzir esse risco, a ferramenta aguarda 10 segundos entre lotes automáticos e interrompe a execução quando a API retorna erro de limite.

Origem e Tipo dos XMLs

A API Nacional disponibiliza documentos do Padrão Nacional da NFS-e. Notas de municípios que ainda usam layouts próprios ou integrações legadas podem não aparecer nesse barramento. O retorno pode incluir NFS-e e eventos associados, como cancelamento.

Certificado Digital e Vínculos

O mTLS exige autenticação com certificado digital ICP-Brasil. Esta versão local aceita certificado A1 em arquivo .pfx ou .p12. A consulta só retorna documentos vinculados ao CNPJ autorizado pelo certificado ou informado para consulta.
